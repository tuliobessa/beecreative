Core Technologies

In order for us to create a website some Core Technologies are used a and we can devide them in:

1.Languages:

In order for you to access a webpage you will need a browser(chrome, safari, IE,etc) which a are a piece of software that looks up the IP number of the requested website and is able to receive data that it can render into the websites that we see.

The data we received from the server usually consist of 3 different files:

1.1.HTML:

It stands for HyperText Markup Language. This technology is responsible for the structure of your website that is mainly used to add elements like text, images, buttons, links or anything involving the main structure of the web page. It is based on the Markup used to be put into manuscripts where editors would markup the manuscript to specify changes to the author or specify structure or layout to the publishers.

HTML works in the same way where you specify layout and structure of you website by using HTML tags. Here is an example of a HTML Tag

The character in between the 2 brackets indicates the tag’s purpose, also known as element ( in this case a paragraph). Most tags have an opening and closing tag (which has a forward slash before the element) and in between them is the content.

1.2.CSS:

CSS(Cascading Style Sheets)are files responsible for styling your website including the design, layout and variations in display for different devices and screen sizes. It changes aspects of your website such as colour, fonts, position, sizes, etc. Here is an example of the structure of a CSS styling

The selector targets which html element we are changing. The property is the way you can style a given HTML element And the property values chooses one of the many appearances for a given property.

1.2.JAVASCRIPT:

Responsible for all the behaviour on your website. It is a scripting language used to create and control dynamic website content. It allows the elements of your website to have action such as send an e-mail when you click a button, your navigation bar to have a dropdown menu, the autocomplete text suggestions, animated graphics, slideshows and so on.

Javascript is a language use to automate processes that users would otherwise need to execute on their own. Used to add interactivity to website, develop mobile applications, create web browser based games and even back end development. 1.Frameworks and Libraries:

Even though developers might use the terms framework and library interchangeably there is a difference. Both of them are code written by someone that is basically reusable code to make your life easier. The technical difference would be on something called inversion of control.

On libraries you are in charge of the flow of the application. You choose when and where to call the library. And when you use a framework, the framework is in charge of the flow. It provides some places for you to plug in your code, but it calls the code you plugged in as needed. Libraries is a collection of reusable code that can be used in our code when we need it.

Most languages have their own libraries but you can write you own custom library if you want. Example of libraries are Jquery, Bootstrap, Yahoo User Interface (YUI). Examples of frameworks are Angular, Ruby on rails, YII, Meteor JS
