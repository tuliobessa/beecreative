1.1.HTML:

It stands for HyperText Markup Language. This technology is responsible for the structure of your website that is mainly used to add elements like text, images, buttons, links or anything involving the main structure of the web page. It is based on the Markup used to be put into manuscripts where editors would markup the manuscript to specify changes to the author or specify structure or layout to the publishers.
